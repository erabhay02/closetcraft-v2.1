# 🏠 ClosetCraft — Custom Closet Designer

Design your perfect closet storage system with AR measurement, drag-and-drop layout, material selection, and PDF export.

**Platform:** iOS + Android (React Native / Expo)  
**Version:** 2.1.0  
**Target Users:** Homeowners & DIY enthusiasts

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📷 **AR Camera Measurement** | Measure walls using ARKit (iOS) or ARCore (Android). Falls back to photo-based measurement on unsupported devices. |
| 📐 **Manual Measurement** | Enter dimensions with sliders (inches or cm) |
| 🎨 **Drag & Drop Designer** | 2D canvas with 12 storage components — shelves, hanging rods, drawers, shoe racks, cubbies, baskets, hooks, valet rods, jewelry trays, hampers |
| 🪵 **10 Materials** | White Melamine, Natural Oak, Dark Walnut, Maple, Cherry, Espresso, Slate Gray, Almond, Chrome Wire, White Wire |
| 🧊 **3D Preview** | Interactive 3D view with orbit, front, top-down, and walk-through modes |
| 💾 **Save & Load** | Unlimited designs stored locally on device |
| 📄 **PDF Export** | Professional design sheets with layout diagrams, dimensions, and component lists |
| ↩️ **Undo / Redo** | 50-step history for the designer |
| 📳 **Haptic Feedback** | Tactile feedback on snap, place, and delete |

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Expo CLI: `npm install -g expo-cli`
- EAS CLI: `npm install -g eas-cli`
- Expo Go app on your phone

### Run in Development

```bash
cd closetcraft
npm install
npx expo start
```

Scan the QR code with Expo Go (Android) or Camera (iOS).

### Build for App Stores

```bash
# Configure
eas build:configure

# Build iOS
eas build --platform ios --profile production

# Build Android
eas build --platform android --profile production

# Submit
eas submit --platform ios
eas submit --platform android
```

### Generate App Icons

```bash
npm install sharp   # one-time dependency
node scripts/generate-icons.js
```

## 🏗️ Architecture

### AR Measurement Flow
```
User taps "Camera Measure"
    │
    ├─ ARRouterScreen checks device capability
    │     ├─ ARKit/ARCore available → NativeARScreen
    │     └─ Not available → ARMeasureScreen (photo-based)
    │
    └─ Measurements → Designer
```

### Data Persistence
```
AsyncStorage (on-device)
    ├── designs[]        → SavedDesignsScreen
    ├── settings{}       → SettingsScreen
    └── lastDesignId     → Auto-resume
```

### Measurement Accuracy
| Method | Accuracy | Requirement |
|--------|----------|-------------|
| ARKit (iOS) | ±0.5 in | iPhone 6s+ |
| ARCore (Android) | ±0.5 in | ARCore device |
| Photo + Reference | ±2-3 in | Any camera |
| Manual (tape) | ±0.25 in | Tape measure |
